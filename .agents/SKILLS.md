# Skills de IA Adicionais

Este documento lista "skills" ou "scripts complexos" que os agentes de IA podem usar para atuar neste repositório.

## Skills Disponíveis

> [!NOTE]
> Atualmente não há scripts de skill customizados mapeados nestes diretórios, mas esta seção reserva o espaço para a integração de automações em bash, python ou Node.Js executáveis pelo agente para facilitar testes unitários intensivos ou migração de banco de dados.

### Exemplo de Aplicação Futura

- `skill: generate_mock_patients` -> Script Node/Firebase admin para popular o emulador com milhares de pacientes fictícios LGPD-compliant.
- `skill: audit_tailwind_colors` -> Script para iterar na source verificando se não existe cor hardcoded violando o Material Design 3 tokens.
